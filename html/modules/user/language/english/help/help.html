User Module

Overview

This module allows you to manage users, groups, and roles defined in the default security realm.

Users are entities that can be authenticated. Each user is assigned a unique identity within the realm.

To make it easier to administer a large number of users, users can be organized into named groups.

Groups can in turn be assigned membership in other groups.

Only users logged in as a member of the Webmaster group Administrators group can add, delete, or modify a user, group, or role..

User Management

The following features are provided:

  • Authentication
  • User Registration
  • User Profile Edit
  • User Profile Display
  • Blocks about users
  • User Management
