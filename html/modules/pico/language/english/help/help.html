Overview

This module allows you to manage content and publications workflow with a granular permissions by user or groups.

Settings allow wysiwyg editors with options according to each use group.

pico is a module for static contents based on Duplicatable V3 (D3). Don't forget updating this module after overwritting older version. And you have to "sync" once on upgrading from 1.5x/1.6x Though functionally pico looks a successor to TinyD, pico is made as full-scratched module. TinyD became a deprecated module because pico has been marked as stable. Of course, you can upgrade your contents from TinyD to pico easily.
